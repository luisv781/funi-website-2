how to play emulated games

Update: RetroArch can't be installed on a chromebook, but the website should work fine. You still have to download roms tho.

RetroArch is a program that lets you emulate games from the gba, ds, and a lot of other retro consoles.

You can download RetroArch at https://retroarch.com and you can also play online but it's slower.

when you download it, make sure you are clicking download and not installer

installers are blocked, so make sure you are clicking the right link

After it's done downloading, you will have a .7z file that you might not be able to open

To open it, you will have to download 7-ZipPortable.exe from the apps folder in my google drive

Open 7-Zip after you're done installing it, and use it to open the .7z file you downloaded from the RetroArch website.

Extract/copy the files out of it and put them somewhere like a folder in your downloads folder.

Then you can go to the folder you extracted and you should be able to find a file called "retroarch.exe" in the middle of all those files

Don't open it yet because you won't have any games installed yet


where to download roms for RetroArch

I recommend this website for downloading roms

It's blocked unfortunately, but if you have a portable browser and use a different wifi it should work.

I'll also be uploading requested roms in google drive


After you're done downloading roms, you need to add cores to RetroArch

cores are like separate console emulators that you'll need to install before you can use a rom

Open retroarch.exe and go to Online Updater. You'll need to be connected to your own wifi or you will get an error.

Click on Core Downloader and a list of consoles should show up

Download the consoles that you want to emulate and after you're done, press x to go back.

You can now go to Load Content to open the roms that you downloaded

Press z to choose a folder and x to go back. Your downloads folder is in the D:\ drive.

To exit a game, press f1 to open the quick menu. From there you can also save and load state

you should also favorite the game, so you can find it more quickly.
